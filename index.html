Technical Challenges

Translating the conceptual design into a functional tool required solving key logical hurdles:

  • Proximity Detection: Implementing Euclidean Distance to ensure the M and D keys target the correct vertex accurately.
  • State Management: Controlling transitions between "Idle" and "Active" modes to prevent accidental data loss.

Heuristic Evaluation

Testing against HCI principles to ensure a smooth user experience:

  • System Visibility: The canvas provides real-time feedback during mouse movements.
  • Error Recovery: The R (Refresh) key allows users to reset the workspace instantly.
  • Efficiency: Keyboard interrupts provide a fast path for experienced users.
