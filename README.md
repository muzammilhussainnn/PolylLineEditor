## Challenges and Confusions
* **State Management:** Managing the transition between "Idle," "Drawing," and "Moving" modes to ensure keyboard triggers didn't conflict.
* **Selection Precision:** Calibrating the pixel-distance threshold for the 'm' and 'd' keys to make selecting points feel intuitive rather than frustrating.
